# trikey.space
The site for [The Social Makerspace](https://www.trikey.space/)

## How it works
1. fork repository
2. clone repository
3. install npm
4. in the repo `npm i`
5. `npm run start` to localhost
6. `npm run build` to create files in the `dist`

Template is `_includes/layout.liquid`, files is `%filename.md`
